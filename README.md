# UKMM-Bundle-Script
Zsh script for creating a macOS app bundle for [UKMM](https://github.com/NiceneNerd/UKMM), a Breath of the Wild Mod Manager

> [!IMPORTANT]
> The latest version `0.17.0-1` has an issue where it can crash on launch
> Until this is fixed, the script will uses a patched version `0.17.0-2` by [Emiyl](https://github.com/emiyl/UKMM)
> However it UKMM will still think that `0.17.0-1` is newer and will show an update screen on launch
> The script will be updated to use the latest official version when Emyil's fixes have been merged
